# SImple-BookshopManagement-System
 

A **console-based bookshop management** mini project written in **C** for university.
This project was my very first mini project in C programming language for the Structured programming course.
This is basically a very simple system for a bookshop to manage books and has some features which is written below:

## Features
- Add book records
- View all books
- Search books
- Update/Delete books
- Sell books (basic)

## How to Run
Compile and run with GCC:

```bash
gcc main.c -o bookshop
./bookshop
```
